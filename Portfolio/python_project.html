Python

This project used APIs and downloaded datasets to examine the correlation between
stock prices and economic indicators during and immediately following recessions. Python
libraries used included numpy, matplotlib, and pandas.

Surprisingly, the data indicated no reliable correlation between Dow Jones Industrial average
pricing and economic growth/decline as measured by GDP. Our project team hypothesized that DJIA
declines would lead GDP declines, but the numbers said otherwise.
